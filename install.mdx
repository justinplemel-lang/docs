## You need both: skills + plugin

There are two things to install, and you should do both. They serve different purposes:

<Info>
**Individual skills** work in regular Claude chat — when you open a conversation and ask Claude to write an update or prep for a meeting. You upload each skill separately.

**The Federato CS Plugin** works in Cowork — where you set up automations, scheduled tasks, and slash commands like `/salesforce-update`. It bundles all skills together with pre-wired connectors.

Most CSMs use both: the plugin for automated bi-weekly runs in Cowork, and the individual skills for quick one-off requests in chat.
</Info>

## Step 2: Download and install the Federato CS Plugin

This works in Cowork. It includes all three skills above plus pre-wired connectors for Slack, Gmail, Calendar, and Granola.

<Card title="Federato CS Plugin v0.2.4" icon="download" href="https://github.com/justinplemel-lang/docs/releases/download/v1.0/federato-cs-v0.2.4.zip">
  Download .zip — then in Cowork: Customize → Plugins → + Add
</Card>

<Tip>
Already installed the individual skills? You still want the plugin — it adds slash commands (`/salesforce-update`, `/meeting-prep`) and lets you schedule automated runs. The skills and plugin don't conflict with each other.
</Tip>

## How to install a skill

<Steps>
  <Step title="Download the skill .zip file">
    Use the download cards under Step 1 above (Slide generation, Salesforce bi-weekly update, or Meeting prep template).
  </Step>
  <Step title="Open Claude → Customize → Skills">
    In Claude, click **Customize** in the left sidebar, then **Skills**.
  </Step>
  <Step title="Add the skill">
    Click **+ Add** → **Upload a skill**, then select your .zip file (or drag and drop). The .zip must contain a SKILL.md file.
  </Step>
  <Step title="Start a new conversation">
    Claude will automatically use the skill when relevant.
  </Step>
</Steps>
