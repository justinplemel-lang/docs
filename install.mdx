## Two ways to get skills

Skills and Plugins are two different install paths for two different modes of using Claude.

**Path 1: Individual skills (for regular Claude chat)**  
Skills are individual SKILL.md files you upload one at a time into Claude's Project settings. Use this path if you're working in regular Claude chat (not Cowork) or if you only want specific skills. Each skill works independently.

**Path 2: The Federato CS Plugin (for Cowork)**  
The plugin is a .plugin bundle you install through Cowork. It packages all three skills together with pre-wired connectors (Slack, Gmail, Calendar, Granola) and slash commands (`/salesforce-update`, `/meeting-prep`). Use this path if you're using Cowork for automation and scheduled tasks.

## How to install a skill

<Steps>
  <Step title="Download the skill .zip file">
    Use the download cards under Path 1 above (Slide generation, Salesforce bi-weekly update, or Meeting prep template).
  </Step>
  <Step title="Open Claude → Customize → Skills">
    In Claude, click **Customize** in the left sidebar, then **Skills**.
  </Step>
  <Step title="Add the skill">
    Click **+ Add** → **Upload a skill**, then select your .zip file (or drag and drop). The .zip must contain a SKILL.md file.
  </Step>
  <Step title="Start a new conversation">
    Claude will automatically use the skill when relevant.
  </Step>
</Steps>
