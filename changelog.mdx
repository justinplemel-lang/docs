## Changelog

**March 6, 2026** — **Federato CS Plugin v0.2.5**

- Added meeting prep template skill page to docs
- Rewrote Salesforce bi-weekly update page — removed accounts.txt, account setup is now conversational and saved to memory
- Added frontmatter to meeting-prep-template skill for discoverability
- Centralized all downloads on the Install page
- Deleted Automations and Granola.ai pages

**March 5, 2026** — **Federato CS Plugin v0.2.3**

First Cowork plugin for the CS team — bundles Salesforce update generation and meeting prep with pre-wired Slack, Gmail, Calendar, and Granola connectors. Memory-based account configuration and scheduled task support.

**March 2, 2026** — **AI Enablement Hub launched**

Central home for all CS team AI tools, skills, training materials, and feedback. No more scattered resources across Slack, Notion, and Google Drive.

**February 17, 2026** — **Federato Brand Skill - major update**

Overhauled the presentation skill with updated crosshair grid system, strict typography enforcement (Frank Ruhl Libre headlines, Inter body), new two-column halftone layout, and expanded slide patterns.

**February 2026** — **CS AI Assistant - Salesforce update generation**

The Account Mgmt App now generates bi-weekly Salesforce updates from your call notes. Filters last two weeks of notes, formats in ALL CAPS headers with bullet points, ready to paste.

**February 2026** — **Notes tab overhaul**

Complete rebuild of the notes system — ClickUp-style interface, internal vs. external call labels, Granola transcript import with AI-generated notes, and searchable history.

**January 2026** — **Federato Brand Skill created**

First Claude skill built for the team — teaches Claude the full Federato brand system for generating on-brand presentations.

**August 2025** — **Granola.ai rolled out**

AI meeting transcription deployed across Sales, Marketing, and Engineering teams for automatic call documentation.
