# Skill downloads

Mintlify doesn't serve `.zip` on non-Enterprise plans, so the skill zips are distributed via **GitHub Releases**. The hub’s Download buttons point to direct release URLs so users get a one-click download.

**To publish or update the zips:** See [HOW-TO-RELEASE.md](./HOW-TO-RELEASE.md) in this folder — step-by-step (no command line). Use tag **v1.0** for the first release so the existing hub links work.
