## Request a skill or feature

Have an idea for a new skill, automation, or tool improvement? Submit it and it'll go into the backlog.

### How to submit

1. **Request type** — New Claude Skill, Tool Improvement, New Automation, Training Video Request, or Other
2. **Title** — Short description (e.g. "Email tone skill that sounds like me")
3. **Description** — What would this do? How would it help your day-to-day?
4. **Priority** — Nice to have / Would save me time weekly / I need this yesterday
