## AI Enablement Hub helper

A Claude skill that teaches Claude to answer questions about the **AI Enablement Hub**: what skills are, how to install them, where to find the Account App walkthrough, training videos, and how to request a new skill. Claude gives short summaries and the right links instead of generic answers.

<Tip>
**Download:** Use the button below to get the .zip from GitHub (one click; no account needed). Then in Claude: **Customize → Skills → + Add** — upload the SKILL.md file.
</Tip>

<Card title="Download AI Enablement Hub helper (.zip)" icon="download" href="https://github.com/justinplemel-lang/docs/releases/download/v1.0/ai-enablement-hub-helper.zip">
  One-click download. Then in Claude: **Customize → Skills → + Add** — upload SKILL.md. Claude will use it to point people to the right hub pages.
</Card>

## When to use this skill

Install this skill if you want Claude to:

- Answer "what are Claude skills?" or "where's the hub?" with a short summary and links
- Point people to the [Account App](/tools/account-app) walkthrough, [slide generation skill](/skills/federato-brand), [training videos](/training/videos), or [request a skill](/feedback/request-skill)
- Give one accurate link instead of guessing when someone asks for a specific hub page

The skill only activates for AI Enablement Hub–related questions. It does not change how Claude handles unrelated tasks.
