**Claude Skills**

## Skills you can install today

Claude Skills are instruction files that teach Claude specific workflows, brand guidelines, and domain knowledge. Download and add them to your Claude settings.

<Card title="New to skills?" icon="book-open" href="/skills/what-are-skills">
  Read [What are Claude skills?](/skills/what-are-skills) for the full primer — why they matter, what's inside a SKILL.md, skills + MCP, and what makes a good skill.
</Card>

## How to install a skill

<Steps>
  <Step title="Download the skill .zip file">
    Use the download link on the skill page below (e.g. [slide generation skill](/skills/federato-brand) or [AI Enablement Hub helper](/skills/what-are-skills#ai-enablement-hub-helper-optional)).
  </Step>
  <Step title="Open Claude → Customize → Skills">
    In Claude, click **Customize** in the left sidebar, then **Skills**.
  </Step>
  <Step title="Add the skill">
    Click **+ Add** → **Upload a skill**, then select your .zip file (or drag and drop). The .zip must contain a SKILL.md file.
  </Step>
  <Step title="Start a new conversation">
    Claude will automatically use the skill when relevant.
  </Step>
</Steps>

### Visual guide

**1. Open Customize** — In Claude’s sidebar, click **Customize**.

<img src="/images/install-skill/step1-customize.png" alt="Claude sidebar with Customize highlighted" />

**2. Go to Skills** — In Customize, select **Skills** to see your skills and add new ones.

<img src="/images/install-skill/step2-skills.png" alt="Customize page with Skills and Connectors; Skills selected" />

**3. Upload a skill** — Click the **+** button and choose **Upload a skill**.

<img src="/images/install-skill/step3-upload-skill.png" alt="Skills page with Upload a skill option in the menu" />

**4. Select your .zip** — Drag and drop your skill .zip (e.g. `ai-enablement-hub-helper.zip` or `updated-federato-brand-skill.zip`) or click to browse.

<img src="/images/install-skill/step4-select-zip.png" alt="Upload skill dialog and file browser showing skill zip files" />
